Falling Ball:
1. Press right and left arrows to move
2. Fall in between the blank spaces
3. If you get stuck and get squished, the game will end




This game is an undesigned layout which is made with javascript.
It wraps up with html: create two divs: id="character", div id="game".
Open a CSS page and write your own preset for the game: "width, height, color, etc.." and after that create a hole and a block to make the areas where the character must fall. And set them all a relative position since they must addapt with the game pad's position.
And the last general step, for scipt.js: create a math.random for the randomizing of the hole position, so it wont stay in the same place indefinetly. Then add a funtion which triggers the use of key arrows.
