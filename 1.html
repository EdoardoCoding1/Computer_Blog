Jump Cube:
1. Press any Key to jump
2. hop over the obstacles
3. If you touch the obstacles the game will end
This game is a layout for a future project (idea by KnifeCircus YT) and it is a reference and well known ripoff of CHROME DINO, or GEOMETRY DASH.
The file is made up of three different documents of which html. In html you start adding three div's of which: div class="game", div class="character" and div class="block". In your Style dcment you must choose your color and size presets, and with that add a relative position to the game pad. After that, use keyframes and add animation for character, with a jump, to then add a command in html which will refer to the animation: onkeyup="jump()". The hardest step would be creating thetrigger of contact between the block and the character. we create a CHECK DEAD, and then create an alert in case the character hit the block.
